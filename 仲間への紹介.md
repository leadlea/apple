# Mac Status PWA - 仲間への紹介 👥

## 🎯 **このアプリについて**

Mac Status PWAは、**AI搭載の包括的Macシステム監視ツール**です。日本語で自然に会話しながら、システムの詳細情報を取得できる革新的なPWAです。

## ✨ **何ができるの？**

### **🤖 自然な日本語でシステム情報を質問**
- "システムの調子はどう？"
- "バッテリーはあとどのくらい？" 🔋
- "WiFiの信号強度は？" 📶
- "どんなアプリが開いてる？" 🖥️
- "外付けドライブの容量は？" 💾
- "開発ツールの状況は？" ⚙️
- "システムの温度は大丈夫？" 🌡️

### **📊 包括的システム監視**

#### **基本システム情報**
- CPU使用率・コア数・周波数
- メモリ使用量・使用率・総容量
- ディスク使用量・使用率・総容量
- ネットワークI/O統計
- システムアップタイム・ロードアベレージ

#### **詳細システム情報**
- **🔋 バッテリー**: 残量、充電状態、残り時間
- **📶 WiFi**: SSID、信号強度、チャンネル、セキュリティ
- **🖥️ 実行中アプリ**: アプリ一覧、リソース使用量、ウィンドウ数
- **💾 ディスク詳細**: 全パーティション、外付けドライブ、容量詳細
- **⚙️ 開発ツール**: Xcode、Git、Homebrew、Node.js、Python、Docker、VS Code
- **🌡️ 温度・ファン**: サーマル状態、電源情報

### **🎨 インテリジェントな応答システム**
- **キーワード検出**: 確実な情報提供
- **コンテキスト理解**: 質問の意図を正確に把握
- **リアルタイム更新**: WebSocketによる即座の情報反映
- **エラーハンドリング**: 堅牢なフォールバック機能

## 🔧 **クイックスタート**

### **必要なもの**
- **OS**: macOS 10.15+ (Catalina以降)
- **Python**: 3.12+
- **RAM**: 8GB+ (推奨: 16GB+)
- **Storage**: 5GB+ (モデルファイル用)
- **時間**: 約15分

### **インストール手順**
```bash
# 1. リポジトリクローン
git clone <repository-url>
cd mac-status-pwa

# 2. 依存関係インストール
pip3 install -r requirements.txt

# 3. ELYZAモデルダウンロード
mkdir -p models/elyza7b
# ELYZA-japanese-Llama-2-7b-instruct.Q4_0.gguf を配置

# 4. サーバー起動
python3 working_server.py

# 5. ブラウザでアクセス
open http://localhost:8002/fixed
```

### **PWAとしてインストール**
1. ブラウザでアプリを開く
2. アドレスバーの「インストール」ボタンをクリック
3. デスクトップアプリとして使用可能！

詳細は [INSTALL.md](INSTALL.md) を参照。

```bash
# モデルディレクトリ確認
```

```bash
# 利用可能メモリ確認
```
